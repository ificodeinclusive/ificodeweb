OPTIMIZATION
Google PageSpeed 90+ Optimization Guide

(Applied specifically to this page type)

Code & Framework

• Use Next.js (SSR or SSG)
• Enable automatic code splitting
• Remove unused JavaScript and CSS
• Avoid heavy UI libraries unless necessary

Images

• Convert all images to WebP
• Lazy-load images below the fold
• Use responsive image sizes
• Keep hero images under 150 KB

Fonts

• Use 1 font family max
• Preload critical fonts
• Use font-display: swap

CSS & Animations

• Purge unused Tailwind classes
• Avoid large box shadows and blur effects
• Use GPU-friendly animations only

JavaScript

• Defer non-critical scripts
• Remove unused third-party scripts
• Avoid heavy analytics scripts on load

Server & Delivery

• Enable Brotli or Gzip
• Use Cloudflare CDN
• Enable browser caching
• Use HTTP/2 or HTTP/3

Page Structure

• Reduce DOM size
• Avoid nested divs
• Keep above-the-fold content lightweight
